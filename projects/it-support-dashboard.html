IT Support Performance Dashboard

Power BI dashboard analyzing 1,000 IT support tickets (FY2024) to help IT managers monitor SLA compliance, agent workload, and operational performance.

Business Problem

IT managers lacked visibility into SLA performance and ticket trends across clients, categories, and communication channels. The objective was to centralize KPIs into a dynamic dashboard for real-time operational monitoring.

Dataset

• 1,000 IT support tickets • Fiscal Year 2024 • Variables: SLA status, agent, category, client, channel, resolution time

Key Insights Delivered

• Identified SLA breach patterns by category • Analyzed agent workload distribution • Highlighted peak ticket volume periods • Segmented performance by client and channel

Tools Used

Power BI | Data Modeling | DAX | KPI Design
